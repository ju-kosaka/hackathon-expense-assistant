# 経費精算アシスタント - Hackathon MVP

AIがあなたの経費精算を爆速化するWebアプリケーション

## 🚀 クイックスタート

### ローカルで起動

```bash
cd hackathon-app
pip install -r requirements.txt
streamlit run app.py
```

ブラウザで `http://localhost:8501` を開く

- ✅ CSVファイルアップロード
