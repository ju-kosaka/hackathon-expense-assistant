## コンセプト

### なぜこのアプリを作るのか
- 社内メンバー（特に非エンジニア）がAI活用を体験的に学べる場を提供
- ブログ記事を読むだけでなく、実際に手を動かして学ぶ
- 「できないこと」「難しい理由」も明示して、現実的な理解を促進

### ターゲットユーザー
- 総務・営業など、IT知識がない社内メンバー
- AI活用に興味はあるが、何から始めればいいかわからない人
- 実践的なスキルを短時間で習得したい人

### 重要な考え方
1. **100%自動化より、一番つらい部分だけ自動化**
   - 完全自動化は難しい（権限、セキュリティ）
   - 手動操作を残しつつ、時短を実現

2. **プチ仕様駆動開発**
   - PLAN.md: 音声入力で思考をダンプ
   - SPEC.md: Claudeと壁打ちして認識齟齬をなくす
   - TODO.md: タスク管理
   - KNOWLEDGE.md: ハマったことを記録

3. **音声入力の重要性**
   - タイピングでは出せない情報量
   - フィラー混じりでOK
   - 10分で数千文字

4. **モノレポでの雑務管理**
   - 小さな作業を1つのリポジトリで管理
   - Claudeがコンテキストを保持
   - README.mdで手順・ハマりポイントを記録
