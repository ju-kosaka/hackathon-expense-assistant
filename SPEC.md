# 体験型AI学習アプリ - 仕様書

## アプリ概要
AI活用の実践的なスキルを体験を通じて学習できるWebアプリケーション

## ターゲットユーザー
- IT知識がない総務・営業などの非エンジニア
- AI活用に興味があるが、どこから始めればいいかわからない人
- 実践的なスキルを短時間で習得したい社内メンバー

## アプリ名
**目指せ！みのるんマスター！**

## ページ構成

### トップページ（`app.py`）
学習コンテンツの一覧を表示

#### 表示内容
各学習コンテンツをカード形式で表示：
- **タイトル**: 学習コンテンツの名前
- **概要**: 何を学べるか（2-3行程度）
- **学習時間**: 所要時間の目安（例: 5分、10分、30分）
- **難易度**: ⭐⭐⭐（星の数）
- **クリックアクション**: 該当コンテンツページへ遷移

#### メタ学習要素
- 💡 ヒント表示: 「このアプリの学習コンテンツは `contents.yaml` で管理されています」
- GitHubリンク: YAMLファイルを直接確認できる
- 学習効果: データ管理の方法も体験的に理解できる

## 学習コンテンツのデータ管理

### 管理方法
**YAMLファイル（`contents.yaml`）**

理由:
- 読みやすい（非エンジニアでも編集可能）
- コメントが書ける → 学習用の説明を記載
- コンテンツ追加が簡単
- GitでのDiff確認がしやすい

### コンテンツデータ構造（確定版）
```yaml
contents:
  - id: "01"
    title: "経費精算を5分で終わらせる"
    description: "毎月の経費精算をAIに任せて、30分→5分に短縮する方法を体験。CSVをアップロードして、Claudeが自動で勘定科目を分類します。"
    duration: "10分"
    difficulty: 1  # 1=⭐, 2=⭐⭐, 3=⭐⭐⭐
    page: "01_💰_経費精算"  # pagesフォルダ内のファイル名（拡張子なし）
    icon: "💰"  # カードに表示するアイコン
```

**フィールド説明:**
- `id`: ユニークなID（文字列推奨）
- `title`: 学習コンテンツのタイトル
- `description`: 何を学べるか（2-3行）
- `duration`: 所要時間の目安（"XX分"形式）
- `difficulty`: 難易度（1=初級, 2=中級, 3=上級）
- `page`: `pages/`配下のファイル名（拡張子`.py`なし）
- `icon`: カード表示用の絵文字

## 実装の詳細

### ページ遷移の仕組み
```python
# contents.yamlから読み込み
contents = load_contents()

# ボタンクリックでページ遷移
if st.button("学習を始める"):
    st.switch_page(f"pages/{content['page']}.py")
```

### YAMLファイルの読み込み
```python
import yaml
from pathlib import Path

def load_contents():
    yaml_path = Path(__file__).parent / "contents.yaml"
    with open(yaml_path, 'r', encoding='utf-8') as f:
        data = yaml.safe_load(f)
    return data['contents']
```

### 難易度の星表示
```python
def get_difficulty_stars(difficulty):
    """難易度を星に変換"""
    return "⭐" * difficulty
```
